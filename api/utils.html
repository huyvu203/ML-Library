Get the configured logger instance.

ml_library.utils.logger.setup_logger(level='INFO', log_file=None, rotation='10 MB', retention='1 week', format=None)[source]

Configure the logger for the ML library.

Parameters:

  • level (Union[str, int]) – Log level to use. Can be a string or int.

  • log_file (Optional[str]) – Path to the log file. If None, logs to stderr only.

  • rotation (Optional[str]) – When to rotate the log file. Default is “10 MB”.

  • retention (Optional[str]) – How long to keep log files. Default is “1 week”.

  • format (Optional[str]) – Log format string. If None, uses the default format.

Return type:

None
